Data Deletion Instructions for My Application

Important Notice: This application is intended for personal use only and is not customer-facing. I have integrated Facebook Login solely to simplify access to my Facebook page and account, allowing me to manage posts and reply to comments directly from within the application without switching to the Facebook platform.

About Data Collection

The application does not collect, store, or reuse any personal data from Facebook. There is no database or data retention system involved. Facebook login is used purely for authentication purposes to streamline my personal workflow.

As the developer and sole user of this application — and a real estate agent — I confirm that no user data is being stored or shared in any form.

Note: Since no data is collected or stored by this application, there is no data to delete from our systems. However, you can revoke the application's access to your Facebook account at any time by following the instructions below.

How to Remove Facebook Access

If you would like to remove Facebook access to the application, you can do so by following these steps:

  1. Go to your Facebook Account Settings.
  2. Navigate to Apps and Websites.
  3. Find and select the application.
  4. Click Remove to revoke access.
